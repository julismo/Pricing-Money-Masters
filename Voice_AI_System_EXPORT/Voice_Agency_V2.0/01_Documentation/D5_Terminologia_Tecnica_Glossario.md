## GÍRIAS E EXPRESSÕES

### Gírias de Pedidos

| Cliente Diz | Significado |
|-------------|------------|
| Quero ficar alinhado | Quer corte limpo e profissional |
| Tipo aquele ator X | Quer referência visual específica |
| À lá David Beckham | Undercut + barba aparada |
| Moderninho | Fade, degradê, design |
| Como sempre | Repetir último serviço |
| Surpreende-me | Quer sugestão criativa do barbeiro |
| Um corte rápido | Serviço básico sem extras |
| Deixa crescer em cima | Manter comprimento no topo |
| Curto dos lados | Degradê ou fade |

### Expressões de Confirmação

| Expressão | Significado |
|-----------|------------|
| Pois, pois | Concordância |
| Pronto / Tá bem | Fechamento/concordância |
| Tipo... | Hesitação ou comparação |
| Olhe... | Introdução de nova ideia |
| Sabe? | Pedido de confirmação |
