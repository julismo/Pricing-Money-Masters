# Meta-Prompt: Pesquisa de Pricing Voice AI para Barbearias

Use este prompt no Perplexity para obter dados que validem/corrijam o modelo de pricing.

---

## PROMPT PARA COPIAR

```
Preciso de dados concretos sobre pricing de Voice AI/Virtual Receptionist para o nicho de barbearias e salões de beleza em 2025. Responde com fontes e números específicos:

### 1. TETOS MÁXIMOS DE PRICING
- Qual é o preço MÁXIMO mensal que serviços de Voice AI para salões cobram?
- Existe um "price ceiling" padrão na indústria?
- Acima de que valor mensal deixa de fazer sentido vs. contratar humano?

### 2. ECONOMIA DE ESCALA
- Como os provedores (Dialzara, MyAIFrontDesk, Smith.ai, GoodCall) ajustam pricing para volumes altos?
- Há redução de % conforme o volume aumenta? (Ex: 25% para pequeno, 15% para médio, 10% para grande)
- Que thresholds de volume são usados para mudar de tier?

### 3. SETUP FEES
- Qual é a faixa de setup fees para barbearias/salões?
- Há um máximo típico de setup para SMBs?
- Grandes redes/franquias pagam setup diferente?

### 4. CUSTO POR RECEPCIONISTA
- Em Portugal, quanto custa um recepcionista de salão (salário + TSU + subsídios)?
- A partir de quantas chamadas/dia um salão precisa de 2 recepcionistas?
- Existe benchmark de chamadas por recepcionista/hora?

### 5. REDES/FRANQUIAS vs UNIDADE ÚNICA
- Como os provedores de Voice AI diferenciam pricing entre:
  - Barbearia única (1 local)
  - Pequena rede (2-5 locais)
  - Franquia (5+ locais)
- Há desconto por volume de locais?

Prioriza dados de 2024-2025 e fontes como: Zenoti, MyAIFrontDesk, Dialzara, Smith.ai, GoodCall, GivePay AI, Marlie AI.
```

---

## O QUE PROCURAMOS VALIDAR

| Parâmetro | Nossa Suposição | A Validar |
|-----------|-----------------|-----------|
| Manutenção máx | €400/mês | Confirmar se é reasonable |
| Setup máx | €5.000 | Confirmar threshold SMB |
| Economia escala | Sim, existe | Como implementar (% ou flat) |
| Chamadas/recepcionista | ~100/dia | Confirmar |
| Multi-local discount | ? | Existe? Quanto? |
